## Sa GitHub Page na Binubuksan Mo:

1. **Hanapin ang green button na "Code"** (makikita mo ito sa taas ng page)
2. **Click ang "Code" button**
3. **Makikita mo ang URL** sa dropdown (may "HTTPS" tab)
4. **I-click ang copy icon** (ðŸ“‹) sa tabi ng URL
   - O kopyahin mo manually ang URL

## Format ng URL:
```
https://github.com/USERNAME/Menstrual-Cycle-Tracking-App.git
```

## Halimbawa:
Kung ang username mo ay `juan123`, ang URL ay:
```
https://github.com/juan123/Menstrual-Cycle-Tracking-App.git
```

---

## Alternative: I-check ang Browser Address Bar

Tingnan ang address bar ng browser mo. Makikita mo doon ang URL ng repository.

Halimbawa:
```
https://github.com/USERNAME/Menstrual-Cycle-Tracking-App
```

**Important:** I-dagdag ang `.git` sa dulo kapag gagamitin sa Git commands:
```
https://github.com/USERNAME/Menstrual-Cycle-Tracking-App.git
```
